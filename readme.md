# KinoPoisk

Современный веб-сайт для поиска, просмотра и изучения информации о фильмах и сериалах. Проект выполнен на HTML5 и CSS3.

## 🚀 Стек

- HTML5
- CSS3

## ⚡ Быстрый старт

1. Скачайте или клонируйте проект через GitHub.
2. Откройте файл index.html в любом современном браузере.
